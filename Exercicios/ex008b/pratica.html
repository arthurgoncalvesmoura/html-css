Códigos

Quando quisermos adicionar um código de programação por exemplo, é só adicionar a tag CODE.

num = int(input('Digite um número: ')) if num % 2 == 0: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim do programa')

Essa tag faz com que todo o código fique mono-espaçado, ou seja, todas as letras ocupam um mesmo espaço, que melhora a leitura do código.

Tag PRE

Como vimos aqui emcima, temos um código que mostra códigos de programação, porem ficou meio descorganizado, por que a tag CODE colocou tudo em linha reta e não deixou o código do jeito que ele foi escrito originalmente. Porem podemos colocar a tag PRE para deixar o codigo sem nenhuma alteração de espacos, e não alterar sua ,para isso vamos juntar a tag CODE com a PRE para ficar o melhor resultado possivel

num = int(input('Digite um número: '))
        if num % 2 == 0:
            print(f'O número {num} é PAR')
        else:
            print(f'O número {num} é ÍMPAR')
        print('Fim do programa')
